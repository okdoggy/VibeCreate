# VibeCreator Editor

Procreate-inspired web image editor built with Next.js.

## Features

- Floating top toolbar with upload, lasso, pointer, and layers controls
- Non-modal floating layers panel that does not affect canvas layout
- Multi-image upload with one uploaded file = one layer
- Centered transparent stage with wheel zoom
- Pointer-based layer selection and drag movement
- Modern soft-glass UI with Phosphor icons

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Production check

```bash
npm run lint
npm run build
```
